Politique de confidentialité

Dernière mise à jour : mars 2026 — Version 1.0

KidInvest est une application d'éducation financière destinée aux enfants, utilisée sous la supervision d'un parent. Aucune donnée n'est vendue. Le compte parent est nécessaire pour accéder à l'application.

1. Qui sommes-nous

KidInvest est édité par FlowForges, représenté par Joffrey Desforges. Pour toute question relative à vos données personnelles : contact@flowforges.fr

2. Données collectées

Lors de l'utilisation de KidInvest, nous collectons :

  • Compte parent : adresse e-mail et identifiant utilisateur (via Firebase Auth)
  • Profil enfant : prénom de l'enfant, saisi par le parent (base légale : intérêt légitime à personnaliser l'expérience pédagogique, cf. article 6.1.f du RGPD)
  • Données de progression : scénarios joués, choix effectués, score d'actifs/passifs, niveau Kiyosaki atteint
  • Données de paiement : gérées exclusivement par RevenueCat et les stores (Apple, Google) — nous n'accédons pas aux informations bancaires
  • Consentement : date et version des CGU acceptées, enregistrées pour preuve de consentement (RGPD art. 7)

3. Finalités du traitement

  • Fournir le service d'éducation financière et suivre la progression de l'enfant
  • Personnaliser l'expérience avec le prénom de l'enfant
  • Gérer votre abonnement (plan gratuit / Premium famille)
  • Respecter nos obligations légales (RGPD, droit à l'effacement)

4. Protection des données enfants

KidInvest est conçu pour les enfants de 8 à 16 ans, utilisé sous compte parent. Nous appliquons les principes suivants :

  • Seul le prénom de l'enfant est collecté — aucune date de naissance exacte, aucune photo
  • Les données de l'enfant sont associées au compte parent et ne sont accessibles que via ce compte
  • Aucune communication marketing n'est adressée directement à l'enfant
  • La suppression du compte parent entraîne la suppression immédiate et définitive de tous les profils enfants et de leurs données

5. Durée de conservation

Les données sont conservées tant que le compte parent est actif. En cas de suppression du compte depuis l'application (Paramètres → Supprimer mon compte), toutes les données — profils enfants, historique de progression, données de compte — sont supprimées immédiatement et définitivement.

6. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : consulter les données que nous détenons sur vous
  • Rectification : corriger des données inexactes
  • Suppression : demander la suppression de votre compte et de toutes vos données
  • Portabilité : recevoir vos données dans un format lisible
  • Opposition : vous opposer à certains traitements

Pour exercer ces droits : contact@flowforges.fr

Vous pouvez également supprimer votre compte directement depuis l'application (Paramètres → Zone dangereuse → Supprimer mon compte), ce qui déclenche la purge immédiate de toutes vos données.

7. Transferts internationaux de données (LEGAL-005)

Nous ne vendons pas vos données. Nous partageons uniquement avec :

  • Google LLC / Firebase (États-Unis) — authentification, base de données Firestore, hébergement. Firebase est soumis aux clauses contractuelles types (Standard Contractual Clauses, art. 46 RGPD) et au Data Processing Agreement de Google Cloud. Données hébergées dans la région europe-west.
  • RevenueCat Inc. (États-Unis) — gestion des abonnements. Votre identifiant utilisateur (UUID anonyme) est partagé pour associer votre abonnement à votre compte. Encadré par les clauses contractuelles types.
  • Apple / Google — via les stores pour la distribution et la facturation.

Ces transferts hors de l'Union Européenne sont encadrés par des clauses contractuelles types (Standard Contractual Clauses) conformément à l'article 46 du RGPD.

8. Cookies et traceurs

L'application mobile KidInvest n'utilise pas de cookies. Elle utilise le stockage local de l'appareil (SharedPreferences chiffré) pour maintenir votre session et enregistrer votre consentement.

9. Modifications

Cette politique peut être mise à jour. En cas de changement significatif, vous serez notifié via l'application. La date et la version sont toujours indiquées en haut de cette page.

10. Contact

Pour toute question : contact@flowforges.fr
